Add your project

This will guide you on how to add your own project to this site.

If you use GoReleaser, all you need to do is to add an empty YAML file to the GoDownloader tree folder, under the right path.

For example, if your project lives under https://github.com/foo/bar, you may create the YAML file at tree/github.com/foo/bar.yaml.

After that, you just need to open a pull request, and everything will happen magically.

If you do not use GoReleaser, additional steps will be required.

You can see the list of all projects being served here.
